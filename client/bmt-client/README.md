# Beast Mode Todo 🔥

A focus time-tracking app with XP system, streaks, and "Beast Mode" to keep you motivated.

## Features

- ⏱️ Active time tracking with play/pause/stop
- 🏆 XP system — earn points for focused work
- 🔥 Streak counter — keep your momentum going
- 💪 Beast Mode — toggle when you're in the zone
- 📊 Stats — Min/Max/Avg time per task
- 💾 Local storage — your data stays in your browser

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Zustand (state management)
- Base UI

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
